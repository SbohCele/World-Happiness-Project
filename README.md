# World-Happiness-Project
This project analyzes the World Happiness Report 2024 dataset to uncover the key factors influencing global happiness. It includes exploratory data analysis, correlation studies, and a regression model to predict happiness scores using socio-economic indicators such as GDP per capita, social support, life expectancy, and freedom.

Skills Demonstrated:
Data Cleaning & Preprocessing,
Exploratory Data Analysis (EDA),
Data Visualization,
Correlation Analysis,
Machine Learning (Regression),
Model Evaluation

This project shows that while economic factors like GDP strongly influence happiness, true well-being is a combination of health, freedom, and social support.

Key insights:
- GDP per capita is the strongest predictor of happiness
- Social support significantly improves happiness levels
- Healthy life expectancy plays a major role
- Freedom contributes positively but varies across countries
- Happiness is multi-dimensional, not purely economic
